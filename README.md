# Done With It

**Done With It** is an Expo React Native app designed to help users manage their tasks efficiently. This application follows the guidelines provided in the "Code with Mosh" course, showcasing best practices in mobile app development with React Native and Expo.

## Features

- **Task Management**: Add, track, and complete tasks with a clean and intuitive interface.
- **Responsive Design**: Optimized for various screen sizes and devices.
- **Modern Practices**: Built following industry best practices for React Native development.

## Getting Started

To get started with the "Done With It" app, follow these steps:

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (includes npm)
- [Expo CLI](https://docs.expo.dev/get-started/installation/) (install via npm)
